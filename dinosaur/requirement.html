Dinosaur Game Requirement

Introduction

RAWR! Are you ready to build a replica of the Chrome dinosaur game?

If you haven’t heard of the Chrome dinosaur game, let’s first talk about the background. Chrome dinosaur game is a hidden game from Chrome offline mode. You can make the dinosaur jump to avoid trees. If the dinosaur hits the tree, you are screwed!

Next, please spend some minutes try it yourself and see how far you can go. https://littleaprilfool.github.io/Colaroid-Study-Materials/dinosaur/week2-demo/index.html

Requirement

Now, let’s look at the detailed requirements of this game.

The Game Layout

The basic game layout includes the following elements:

  • Game title and instruction
  • Game area where there is a dinosaur and some trees
  • A score board and a restart button

Rules

  • The user clicks on the game area to make the dinosaur jump.
  • The tree moves from the right to the left of the screen and then disappear. After a tree disappear, a new tree would appear at the right of the screen. The size of the tree is randomized.
  • When the dinosaur hits the tree, the game is over. The user can click on the restart button to restart the game.
  • Every time the dinosaur passes a tree, the user win a score. The speed of the next tree would increase 0.25.

Size and Assets Reference

The Game Area

  • Size: 600px * 300px
  • Background image: https://thumbs.gfycat.com/PlayfulWeeklyGuppy-mobile.jpg
  • The dinosaur and the trees are recommended to be positioned above 90px of the bottom border of the game area

The Dinosaur

  • Size: 50px * 50px
  • Background image: https://user-images.githubusercontent.com/5129786/161480083-24a52fd3-26f6-4abe-bcee-70b74071a59c.png
  • Movement
    • Assume that the dinosaur jumps and falls at a constant speed of 3px per frame
    • The dinosaur can jump no higher than 200px above the bottom border of the game area

The Tree

  • Size: The size of the tree should be randomized.
    • width: 10~20px
    • height: 20~40px
  • Movement
    • Initially 2px per frame
    • The speed increase by 0.25px when the dinosaur passes a tree
  • Background image: https://www.tynker.com/projects/images/e20df433a4316a7c308b11da2fcdb32d432598f4/cacti---cacti2.png
    • The background image should repeatedly fill the container
